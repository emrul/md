# Tips & shortcuts

A few things that make MarkdownMD fast.

## Essential shortcuts

| Action | Shortcut |
| --- | --- |
| New document | ⌘N |
| New window | ⌘⇧N |
| Open… | ⌘O |
| Save | ⌘S |
| Toggle source view | ⌘/ |
| Toggle file explorer | ⌘⇧E |
| Find / Replace | ⌘F |
| Bold / Italic / Inline code | ⌘B / ⌘I / ⌘E |
| Link | ⌘K |

## The slash menu

In an editable document, type `/` at the start of a line to insert headings, lists, tables, code blocks, and more — without remembering the Markdown.

## View modes

Switch per-tab between **WYSIWYG**, **Hybrid**, and **Source** from the toolbar. Your preferred default is remembered for new documents.

## Files stay yours

- Documents are plain `.md` files — open them in any other editor any time.
- Edit a file outside MarkdownMD and it reloads automatically; if you have unsaved changes, you get a banner to compare and choose.
- The file explorer (⌘⇧E) browses any folder on disk.

## Make it your own

These examples are read-only, but everything here works in a real document. Press **⌘N** and try it: type a heading, drop in a table, paste some Markdown — it just renders.

Happy writing.
