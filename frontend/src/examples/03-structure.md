# Structure

Organize a document with headings, lists, tables, and quotes.

## Headings

Use `#` through `######` for six levels. In hybrid mode the whole line resizes; the `#` markers show only when you're editing the line. Press **⌘⌥1 / ⌘⌥2 / ⌘⌥3** to set a heading level.

## Lists

Bullet list:

- Groceries
  - Coffee
  - Oat milk
- Errands
  - Post office

Ordered list:

1. Preheat the oven
2. Mix the batter
3. Bake for 25 minutes

Task list — check items off as you go:

- [x] Install MarkdownMD
- [x] Open the examples
- [ ] Write your first note
- [ ] Tell a friend

## Tables

| Feature | Status | Notes |

| Live tables | ✅ | resize + align |
