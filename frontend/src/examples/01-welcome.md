# Welcome to MarkdownMD

A fast, native Markdown editor that shows your text **rendered** while you write — and reveals the raw Markdown the moment your cursor enters a line.

These tabs are a quick tour. They're **read-only**, so click around freely — you can't break anything. Open them any time from **Help → Examples**.

## How hybrid editing works

Put your cursor in this paragraph. The `#`, `**`, and other markers appear so you can edit them; click away and they hide again, leaving the rendered result. That's *hybrid mode* — the best of a source editor and a live preview, in one place.

Try the three modes from the toolbar (or **⌘/** to peek at the raw source):

- **WYSIWYG** — pure rendered text, no markers ever
- **Hybrid** — rendered at rest, raw Markdown where your cursor is
- **Source** — the plain Markdown, like a code editor

## What's in the other tabs

1. **Formatting** — bold, italic, links, inline code
2. **Structure** — headings, lists, task lists, tables, quotes
3. **Code & math** — fenced code blocks and LaTeX math
4. **Diagrams** — Mermaid flowcharts and more
5. **Tips & shortcuts** — get fast

> Your files are always plain Markdown on disk. MarkdownMD never locks them into a proprietary format — what you write is exactly what's saved.

When you're ready, create a new document with **⌘N** and start writing.
